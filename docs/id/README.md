## Filosofi

Panduan ini dibangun dari sesi nyata — bukan teori. Setiap pola di sini sudah diuji di project sungguhan.

**Prinsip utama**: Kiro bekerja paling baik ketika kamu memberinya struktur. Semakin banyak konteks yang kamu berikan (steering files, specs, agents), semakin baik hasilnya.
