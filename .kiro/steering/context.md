# Project Context — kiro-guide

## Identitas
Gunakan identity **sandikodev** untuk semua operasi git/gh.

Remote: `git@github-sandikodev:sandikodev/kiro-guide.git`

## Project

Community documentation for Kiro IDE, Chat, and CLI.
Bilingual: English (`docs/en/`) + Bahasa Indonesia (`docs/id/`).
Open to all language contributions.

## Struktur

```
docs/
  en/    # English docs
  id/    # Docs Bahasa Indonesia
examples/ # Real project examples
.kiro/   # Kiro config for this repo
```

## Konvensi

- File naming: kebab-case, numbered for ordering (`01-getting-started.md`)
- Each language folder has its own README.md as table of contents
- Screenshots go in `docs/{lang}/assets/`
- Commit: conventional commits — `docs:`, `fix:`, `feat:`, `i18n:`

## Kontribusi

Lihat CONTRIBUTING.md — semua bahasa disambut.
